EXCLUDED FROM PUBLIC API DOCS
==============================

These controllers serve HTML rendering and should NOT appear in
the public API documentation (marked as "internal" tag):

  - jobs/controllers.py:JobRenderController
  - nominations/controllers.py:NominationRenderController
  - pages/controllers.py:PageRenderController
  - search/controllers.py:SearchRenderController
  - sponsors/controllers.py:SponsorRenderController


BENEFITS OF THIS STRUCTURE
===========================

✓ Reduced cognitive load (50 grouped tags vs 100+ fragmented)
✓ Logical developer journey (auth → users → content → features)
✓ Related resources adjacent (Event → Calendar → Category)
✓ Admin isolated and deprioritized
✓ Internal render controllers hidden
✓ Title Case spaces (readable in Scalar UI)
✓ Follows REST resource hierarchy
✓ Scalable for future growth
✓ Better API discovery
